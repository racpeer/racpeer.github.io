About Me

Hai.. Saya Aulia Rachman, lahir pada tahun 1984 di Karawang, Saya pernah menempuh pendidikan di Politeknik Negeri Bandung dan STMIK Jawa Barat, saat ini bekerja di SMKS TI Muhammadiyah sebagai Kepala Program Studi Pengembangan Perangkat Lunak dan Game, pada Konsentrasi Keahlian Rekayasa Perangkat Lunak dan Kepala Program Studi Teknik Komputer dan Jaringan Telekomunikasi, pada Konsentrasi Keahlian Teknik Komputer dan Jaringan.

Saya memiliki ketertarikan terhadap bidang IT, Programming dan Desain Grafis, dalam pengembangan aplikasi berbasis Desktop atau Web, saya menggunakan bahasa pemrograman Java, sedangkan framework CSS-nya menggunakan Bootstrap, dan DBMS yang biasa digunakan adalah MySQL. Sistem operasi yang saya gunakan adalah Windows atau Linux (Ubuntu Desktop / Server).
